Hy-Vee Aisles CLI application.  Remove the ad tracking!

When more services work like cart and such I will add those as well.

Commands

> item
Search For Item
> add item
Add item that was searched
> code upc
Find and add a UPC code in one command
> quit
exit
> help
Show Help
